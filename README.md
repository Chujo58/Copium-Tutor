## ðŸ›  Installation

### Prerequisites
- Node.js (v18+)
- Python (3.10+)
- npm or yarn

### Clone the repo

```sh
git clone https://github.com/Chujo58/Copium-Tutor.git
cd Copium-Tutor
```

### Run the frontend
```sh
cd notion-lite/
npm install
npm run dev
```

### Run the backend
```sh
cd notion-lite/backend
pip install -r requirements.txt
uvicorn main:app --reload
```
